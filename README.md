# DATA01 Executive Search Analytics Assignment

Dieses Repository enthält die Arbeitsgrundlage für das Assignment im Modul Data Analytics.

## Ziel
Entwicklung eines datengetriebenen Akquise-Unterstützungsansatzes im Executive Search auf Basis historischer Sales-Daten auf Debitor-Ebene.

## Struktur
- `notebooks/` enthält die analytischen Notebooks
- `data/raw/` enthält lokale Rohdaten und wird nicht versioniert
- `data/processed/` enthält lokal erzeugte Zwischenergebnisse und wird nicht versioniert
- `docs/notes/` enthält methodische Notizen und Projektdokumentation
- `src/` ist für spätere Hilfsfunktionen vorgesehen
